# Diplom_3
## Тестирование веб-приложение Stellar Burgers
###
| **Использованные библиотеки** |
|-------------------------------|
| java 11                       |
| junit 4.13.2                  |
| maven 3.9.6                   |
| selenium 3.141.59             |
| webdrivermanager 5.6.2             |
| maven-surefire 2.22.2         |
| allure-maven 2.15.0           |

#### Запуск тестов
```
mvn clean test
```


##### Создание отчета Allure
```
mvn allure:serve
```
